# Career Guidance and Placement Prediction System

A Full Stack Development project that predicts whether a student is likely to be placed based on their academic performance.

The system uses a **Machine Learning model** integrated with a **Node.js backend**, **MongoDB database**, and a **web frontend**.

---

## 📌 Features

- Predict student placement using machine learning
- Input academic scores (SSC, HSC, Degree)
- Backend API built with Node.js and Express
- Machine Learning model built using Python
- MongoDB database to store prediction data
- Simple user-friendly web interface

## ⚙️ How to Run the Project

### 1 Start MongoDB
mongod

### 2 Run Backend
cd backend
node server.js


### 3 Run Machine Learning Model
cd ml_model
python app.py


### 4 Open Frontend

Open `frontend/index.html` in a browser.

---

## 📊 How the System Works

1. User enters academic marks in the frontend
2. Data is sent to the Node.js backend
3. Backend calls the Python ML model API
4. ML model predicts placement
5. Result is returned to the frontend
6. Prediction data is stored in MongoDB
